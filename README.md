## Getting Started

Install dependencies:

```bash
# install node modules
npm install
```

Run the server:

```bash
# dev server
npm run dev

# prod server
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
